# Sudoku Game

[简体中文](README.md) | English

An elegant, interactive web-based Sudoku game with multiple difficulty levels and a user-friendly interface.

## Features

- 🎮 Three difficulty levels: Easy, Medium, and Hard
- 🎯 Automatic generation of valid Sudoku puzzles
- ✅ Real-time error checking and complete solution validation
- 🎨 Modern, responsive design that adapts to various devices
- ⌨️ Keyboard input support
- 🔄 Smooth animations and visual feedback

## Technology Stack

- HTML5
- CSS3 (using modern CSS features like Grid layout, Flexbox, and CSS variables)
- Vanilla JavaScript (ES6+)
- Responsive design

## How to Use

1. Clone or download this repository
2. Open the `index.html` file in your browser
3. Select a difficulty level and click "New Game" to start
4. Click on empty cells and use your keyboard to input numbers (1-9)
5. Use the Backspace or Delete key to clear a cell
6. Click the "Check" button to validate your solution when finished

## Game Rules

Sudoku is a logic-based game with the objective of filling a 9×9 grid with digits 1-9 so that each row, column, and 3×3 subgrid contains all digits without repetition.

## Development Features

- Uses a backtracking algorithm to generate valid Sudoku puzzles
- Implements a Sudoku solver to verify unique solutions
- Uses CSS Grid layout to create the Sudoku grid
- Smooth CSS transitions and animation effects
- Fully responsive design that adapts to mobile devices
